# Procès verbaux aubois

Liste détaillée des procès verbaux. Le but est de pouvoir chercher les
différentes décisions prises sur une thèmatique donnée.

```sh
./main.ts
```

Le format utilisé est le suivant:

- Une ligne commencant par `===` sert à indiquer le titre d'une section, la
  ligne sera converti en une balise HTML `h2`;
- Un item: un identifiant (par exemple `2.1`), une espace et le contenu de
  l'item;
- Une ligne vide qui sera ignorée, à utiliser pour séparer ds groupe d'item.
